# 快速开始

## 环境要求

- Node.js >= 18.0
- Git
- Docker (可选)

## 安装步骤

### 1. 克隆仓库

```bash
git clone https://github.com/jiang0233/project-wiki.git
cd project-wiki
```

### 2. 安装依赖

```bash
npm install
```

### 3. 配置环境变量

```bash
cp .env.example .env
# 编辑 .env 文件，配置必要参数
```

### 4. 启动项目

```bash
# 开发模式
npm run dev

# 生产模式
npm run build
npm start
```

## 验证安装

访问 http://localhost:3000 查看项目是否正常运行。
